# Goal

## Objective

Define the full product architecture for Enterprise Policy RAG, including a polished but compact UI surface suitable for portfolio demonstration.

## Scope

In scope:

- Reference review for enterprise search, RAG knowledge management, retrieval testing, admin, and insights patterns.
- End-to-end product architecture with backend, frontend, data, provider, and verification boundaries.
- Essential screen set for portfolio demo.
- Phase plan that reorders implementation around the full architecture before additional coding.

Out of scope:

- Copying proprietary UI layouts exactly.
- Building frontend code in this run.
- Adding OpenAI API integration.
- Expanding into on-premise deployment.

## Exit Criteria

- A design document exists under `docs/internal/design/`.
- README/TODO/project tracking reference the new architecture-first direction.
- This run records changed files, decisions, and verification.

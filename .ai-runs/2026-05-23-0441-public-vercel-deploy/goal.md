# Goal

Publish the prepared static read-only demo to a public Vercel URL and verify the deployed page.

## Scope

- Use the existing static demo build configuration.
- Authenticate Vercel CLI if required.
- Deploy production demo.
- Verify the deployed Operations route renders the expected UI.
- Update project documentation with public URL and deployment metadata.

## Out of Scope

- Git remote setup.
- Vercel Git integration automation.
- Backend/API hosting.
- Production auth/SSO.

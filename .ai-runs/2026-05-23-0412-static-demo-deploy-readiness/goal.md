# Goal

Prepare a public read-only fake-provider demo path that can be deployed without API keys, Docker, or a backend process.

## Scope

- Add a deterministic static demo mode for the React UI.
- Keep local live API mode unchanged.
- Add build/preview verification for the static demo bundle.
- Document deployment options and current limitations.

## Out of Scope

- Storing secrets or running paid external AI calls.
- Production auth/SSO.
- Backend public hosting with PostgreSQL.
